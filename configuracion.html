🔑 Credenciales de Trello

📋 Pasos para obtener tus credenciales:
  1. Abre en tu navegador: https://trello.com/app-key (debes estar con sesión iniciada en Trello)
  2. En esa página verás tu API Key — es una cadena de ~32 caracteres. Cópiala.
  3. En la misma página, haz clic en el enlace que dice "Token" (junto a la key) → aprueba el acceso → copia el token largo que aparece (~64 caracteres).
⚠️ Error "invalid key": asegúrate de copiar el API Key (no el "Secret"). El Key son ~32 caracteres alfanuméricos en minúsculas.
